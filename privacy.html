Privacy Policy

Last Updated: March 2025

1. No Data Collection

JSON Architect operates as a client-side application. When you format, validate, or minify JSON data, all processing occurs locally within your web browser's memory. Your JSON data is never sent to our servers or any third-party servers.

2. Local Storage

We use your browser's Local Storage to save your preferences, such as:

  • Theme (Light/Dark mode)
  • Language preference (English/Spanish)

This data stays on your device and is not tracked by us.

3. Analytics

This website does not use cookies for tracking, advertising, or analytics purposes.

4. Contact

If you have any questions about this privacy policy, please contact us via our GitHub repository.

Política de Privacidad

Última Actualización: Marzo 2025

1. Sin Recopilación de Datos

JSON Architect funciona como una aplicación del lado del cliente. Cuando formateas, validas o minificas datos JSON, todo el procesamiento ocurre localmente en la memoria de tu navegador web. Tus datos JSON nunca se envían a nuestros servidores ni a servidores de terceros.

2. Almacenamiento Local

Utilizamos el Almacenamiento Local de tu navegador para guardar tus preferencias, como:

  • Tema (Modo Claro/Oscuro)
  • Preferencia de idioma (Inglés/Español)

Estos datos permanecen en tu dispositivo y no son rastreados por nosotros.

3. Analíticas

Este sitio web no utiliza cookies para fines de seguimiento, publicidad o analíticas.

4. Contacto

Si tienes alguna pregunta sobre esta política de privacidad, contáctanos a través de nuestro repositorio de GitHub.
